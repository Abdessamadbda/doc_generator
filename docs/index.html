OfferController

The OfferController class handles HTTP requests related to offers, including processing forms, sharing on LinkedIn, and uploading CVs.

Author: Unknown Author

Version: 1.0.0

Last Modified: Unknown

@PostMapping/exchange-code-for-access-token

POST
Exchanges an authorization code for an access token using LinkedIn's API. @param code Authorization code @return ResponseEntity containing the access token

@PostMapping/share-on-linkedin

POST
Shares content on LinkedIn using the provided access token. @param content Content to share @param accessToken LinkedIn API access token @return ResponseEntity containing the LinkedIn API response

@PostMapping/submit

POST
Submits a job offer after validating the JWT token. @param authorizationHeader JWT token @param formData Form data as key-value pairs @return ResponseEntity indicating success or failure

@PostMapping/file

POST
Handles uploading CV files and extracts text from them. @param cvFiles Array of CV files @return ResponseEntity containing extracted text from the CVs

UserController

This class serves as the main REST controller for handling user-related operations such as login, token verification, and saving user data.

Author: Unknown Author

Version: 1.0.0

Last Modified: Unknown

@PostMapping/login

POST
Endpoint to authenticate a user and generate a JWT token. @param loginRequest The login credentials (email and password). @return A response containing the user details and a JWT token, or an unauthorized response if validation fails.

@GetMapping/verify

GET
Endpoint to verify the validity of a JWT token. @param authorizationHeader The "Authorization" header containing the JWT token. @return A JSON response indicating whether the token is valid or not.

@PostMapping/user/save

POST
Endpoint to save a user to the database. @param user The user entity to be saved. @return The saved user entity, or an internal server error if saving fails.
